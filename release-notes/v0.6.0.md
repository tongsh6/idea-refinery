# IdeaRefinery v0.6.0

## Highlights

- Add timeline filtering options to `refinery observe`.
- Support step-level and event-type filtering for faster incident triage.
- Support last-N view to focus on the most recent events.

## CLI Enhancements

- `--step <step>`: filter by stage (`run`, `draft`, `review`, `edit`, `gate`).
- `--event-type <type>`: filter by event type.
- `--limit <N>`: show the latest N matched events.
- Existing selection (`--run-id` and `--latest`) remains available.
