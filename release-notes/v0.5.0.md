# IdeaRefinery v0.5.0

## Highlights

- Add structured run timeline events persisted to SQLite (`run_events`).
- Emit stage lifecycle and gate decision events during orchestration.
- Add `refinery observe` command to print timeline tables for a run.
- Add `refinery observe --latest` to inspect the latest run quickly.

## Observability Improvements

- New event types: `run_started`, `run_completed`, `run_failed`, `stage_started`, `stage_completed`, `stage_failed`, `gate_decision`.
- Event payload includes key execution context such as duration, decision, score, blocking count, cost, and provider details.
- Timeline data can now be replayed from database without parsing raw model output.
